Selectores, box model, estilos básicos

**CSS**, que significa "**Cascading Style Sheets**" (**Hojas de Estilo en Cascada**), es un lenguaje utilizado en el desarrollo de sitios web para dar estilo y diseño a las páginas **HTML**. Mientras que HTML se encarga de definir la estructura y el contenido de una página web, CSS se utiliza para controlar cómo se ve esa estructura y cómo se presentan los elementos en la página. Piensa en una página web como un libro. **HTML** sería como el contenido del libro: el **texto**, los **párrafos**, las **imágenes**, **etc**. Sin embargo, un libro también necesita ser presentado de **manera atractiva** y **fácil** de **leer**. Aquí es donde entra **CSS**. **CSS** sería como el **diseño** del **libro**: el tipo de **letra**, el **tamaño** **del** **texto**, los **colores de fondo**, el **espaciado entre** **párrafos**, y así sucesivamente. Entonces, **CSS** es útil en los sitios web porque permite que los desarrolladores web controlen y personalicen la apariencia visual de sus páginas. Esto les da la capacidad de crear sitios web hermosos y atractivos que sean fáciles de navegar y agradables de ver para los usuarios. Sin CSS, todas las páginas web se verían iguales y serían mucho menos interesantes y atractivas.
